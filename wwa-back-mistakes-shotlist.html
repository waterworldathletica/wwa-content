— PULL UP —
❌ Knees swinging for momentum → Henry kipping, fast, sloppy
✅ Core engaged, heels elevated, pelvis forward → Ash corrects Henry's body position
❌ Pulling into shoulders, poor range → Arms never fully extend, shoulder dominant
✅ Full extension, internal elbow rotation, lean back → Slow, lat-led, 2 perfect reps

— LAT PULLDOWN —
❌ Too upright, wrists bending, momentum → Henry rocking, fast, no range
✅ Proper hand width, internally rotated elbows → Ash adjusts grip and elbow path
❌ No range of motion → Partial reps, no stretch, no contraction
✅ Full range — overhead to clavicle, slow negative → Braced core, lat-led

— SEATED CABLE ROW —
❌ A-frame attachment, jerking, arms only → Fast, chaotic, no stretch
✅ Wide attachment, full protraction, low elbows → Ash sets the start position
❌ No contraction, no pause → Movement stops short, no retraction
✅ Full retraction, pause, elbows scoop to side pockets → Slow controlled return

— FACE PULL —
❌ Leaning back, no core, fast reps → Henry using bodyweight momentum
✅ Braced core, elbows wide and high → Ash corrects stance and elbow path
❌ No range, no hold → Rope barely reaches face, no contraction
✅ Full stretch, overhead finish, 1.5 second hold → Slow exhale on return

Wide portrait — Ash, back visible, lats flared

The one thing that makes or breaks this shoot: the pacing contrast has to be extreme. Henry's bad reps should look frantic and exhausting. The corrected reps should look almost slow-motion by comparison. That visual contrast is more convincing than any text overlay. If the speed difference isn't obvious, reshoot.

Editing Notes
Edit
Hook to corrections — hard cut, no transition
Hook line ends. Hard cut straight into the first correction. No fade. The abruptness signals the video is shifting gears.
Edit
Text overlays — ❌ red, ✅ green. One at a time.
Every mistake and correction labelled on screen. Red X for the mistake, green tick for the correction. Text appears as the action happens. Never two labels simultaneously.
Edit
Closing shots — no text, no cut
Wide back shot holds 2–3 seconds. Punch in to the pose. No text on either closing shot. The silence after all the corrections is what makes these frames land.
Edit
Grade — WWA palette throughout
Crushed blacks, cooler midtones. Gym lighting runs warm — correct it. The closing back shots especially need to be sharp. Muscle detail has to read clearly.
